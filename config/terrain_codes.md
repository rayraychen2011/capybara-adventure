# 地形編碼對照表

基於 Cupertino 地圖的地形物件編碼系統：

## 地形編碼定義

| 編碼 | 地形類型  | 描述                             | 顏色建議 |
| ---- | --------- | -------------------------------- | -------- |
| 0    | 草地      | 一般開放草地區域                 | 淺綠色   |
| 1    | 森林/密林 | Stevens Creek County Park 森林區 | 深綠色   |
| 2    | 水體      | Stevens Creek 溪流               | 藍色     |
| 3    | 道路      | 一般街道                         | 灰色     |
| 4    | 高速公路  | 85 號、280 號高速公路            | 深灰色   |
| 5    | 住宅區    | 低密度住宅區域                   | 淺黃色   |
| 6    | 商業區    | 商店、辦公區域                   | 橘色     |
| 7    | 公園設施  | 公園內的設施區域                 | 綠色     |
| 8    | 停車場    | 停車區域                         | 淺灰色   |
| 9    | 山丘/高地 | 地勢較高的區域                   | 棕色     |

## 使用說明

- CSV 文件中每個數字對應一個地圖格子
- 可以通過修改數字來改變地形類型
- 建議地圖大小為 40x30 格子（配合遊戲設定）
